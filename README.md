## Starting The Project
1. Clone the repository:
  * ```git clone git@github.com:pedrobianchi90/blogs-api.git```
2. Navigate to the cloned repository folder:
  * ```cd blogs-api```
3. Install the dependencies:
  * ```npm install```
4. Start the project:
  * ```npm start```
